자주하는 질문

  • 댕구루 회원가입하면 모든 스토어의 회원가입도 되는 건가요?

    댕구루의 회원가입과 각 판매자 스토어의 회원가입은 별개로 이뤄지고 있습니다.

    때문에 댕구루 ID로 회원가입하시더라도, 개별 판매자 스토어의 회원가입은 따로 진행해야 합니다!


    tip : [마이페이지]-[스토어 가입 도우미]를 이용하시면 일반 판매자 스토어의 회원가입 시 매번 같은 정보를 입력해야 하는 수고를 덜 수 있어요!

  • 댕구루 계정 탈퇴를 하고 싶어요.

    회원 탈퇴 방법은 [마이페이지]-상단의 [설정]-[회원정보수정]-[회원탈퇴] 버튼을 통해 진행하실 수 있습니다!

    탈퇴 시엔 저장하셨던 즐겨찾기 및 찜한 아이템의 데이터, 주문/배송내역은 모두 삭제되어 복구가 불가하니, 이 점 이용에 참고 부탁드립니다.

  • 스토어 가입 도우미는 뭔가요?

    [스토어 가입 도우미]는 개별 판매자 스토어에 가입해야 하는 번거로움을 덜어드리고자 댕구루에서 제공하는 기능으로, 판매자 스토어 가입 시 필요한 많은 정보들을 미리 '저장'해두었다가 자동으로 내용을 '불러오기' 기능입니다!


    * 해당 기능은 각 판매자 스토어의 "회원가입"시에만 이용하실수 있는 기능으로, 개별 스토어의 '비회원 주문시에는 적용되지 않는 점 이용에 참고해 주시기 바랍니다.

  • 주문/배송 화면에서 주문내역이 표시되지 않아요.

    [마이페이지]-[주문/배송]에서 표시해드리는 주문 건의 경우, 주문을 완료한 마지막 페이지의 정보를 불러오고 있어요. 만약 해당 영역에서 주문내역이 확인되지 않는 사유는 아래의 경우와 같습니다!


    주문/결제가 정상적으로 완료되었지만,

    1) 주문'완료' 페이지까지 확인하지 않은 경우

    2) 주문/배송 기능을 아직 지원하지 않는 스토어일 경우


    *1),2) 번에 해당한다면 판매자 스토어의 홈페이지에서 직접 주문건을 확인해주세요!

  • 상품을 교환/반품 하고 싶은데요.

    상품 취소/교환/반품의 의무와 책임은 해당 스토어에 있으므로, 해당 스토어에 요청하시면 됩니다.

  • 주문내역을 모르고 삭제했는데, 내역을 복구할 수 있나요?

    주문내역의 경우 민감한 개인정보(주소, 연락처 등)가 담겨져 있기 때문에 휴대폰 자체에만 저장이 됩니다.

    직접 주문내역을 삭제하셨거나 휴대폰을 변경 혹은 앱을 재설치하신 경우 해당 주문정보는 삭제 처리되며, 복구는 어렵습니다.

    삭제된 주문건은 주문하신 스토어에서 확인해주세요!

  • 주문내역을 모르고 삭제했는데, 주문도 저절로 취소 되는 건가요?

    댕구루 내의 주문내역의 경우, 사용자 분들의 편의를 위해 댕구루 내에서 결제정보 및 배송 현황을 "표시"해드리는 것으로 앱 내에서 주문 정보를 삭제하시더라고 실제 주문이 취소되거나 주문 상태가 변동되지 않습니다.

    댕구루 내의 삭제된 주문 건의 상세 내용은 해당 스토어로 이동하여 확인 부탁드립니다!

  • 댕구루에 원하는 스토어를 추가하고 싶어요.

    댕구루에 새로운 스토어 추가를 원하실 경우, [마이페이지]-[앱 문의/건의하기]에 접수를 해주세요.

    영업일 기준, 관리자의 신규 스토어 검토 후 승인처리가 완료되면 스토어는 등록이 됩니다.

  • 댕구루의 쇼핑 알림/이벤트 등 다양한 소식을 받고 싶어요.

    [마이페이지] - [설정] - [푸시 수신 동의]를 켜주세요.

    이후 기기의 알림 설정에서 댕구루 앱 푸시 수신을 켜주시면 댕구루의 다양한 쇼핑/이벤트 소식을 발 빠르게 받아보실 수 있습니다

    참고) 기기의 설정 > 알림 > '댕구루' 앱 > 알림 허용 켜기

  • 찜해둔 즐겨찾기 및 내 상품들(데이터)이 갑자기 없어졌어요.

    즐겨찾기 스토어 및 내 상품의 찜해둔 상품들이 갑자기 사라진 경우, [마이페이지]-[앱 문의/건의하기]를 통해 댕구루 고객센터로 아래의 정보를 전달해주세요!


    1) 댕구루 ID (마이페이지에서 확인 가능)

    2) 자세한 현상 설명

  • 앱이 자꾸 튕겨요.

    먼저 댕구루 앱 버전과 기기의 OS 버전이 최신인지 확인해주세요!

    업데이트 이후에도 같은 문제가 발생될 경우, [마이페이지]-[앱 문의/건의하기]로 아래의 정보를 전달해주시면 빠르게 확인 도와드리겠습니다.


    1) 댕구루 ID (마이페이지에서 확인 가능)

    2) 오류 화면의 스크린샷과 함께 상세한 내용 설명

  • 특정 스토어에만 들어가면 상품 로딩이 안돼요.

    판매자 스토어 홈페이지 접속 시 쌓이는 웹데이터로 인해 방생될 수 있으며, 이런경우 [마이페이지]-[앱 문의/건의하기]로 아래의 정보를 전달해주세요! 해결방안에 대해 안내해드리겠습니다.


    1) 댕구루 ID (마이페이지에서 확인 가능)

    2) 오류 화면의 스크린샷과 함께 상세한 내용 설명

  • 결제하려다 오류가 났어요.

    해당 스토어 판매자에게 문의하여 확인 바랍니다.

  • 댕구루에서 본 상품의 가격이 스토어로 들어가면 달라요.

    판매자가 스토어 홈페이지에서 상품의 가격정보를 변경하였지만 해당 정보가 정상적으로 앱에 반영되지 않았을 수 있습니다.

    [마이페이지]-[앱 문의/건의하기]를 통해 해당 상품 정보(스토어명, 상품 이름)을 보내주시면 신속히 수정해드릴 수 있습니다!

    상품의 정상 가격은 스토어의 홈페이지에서 판매되고 있는 실제 가격을 통해 확인 부탁드립니다.

  • 상품을 보고 싶은데 유효하지 않은 상품(URL, 연결불가)이라고 떠요.

    판매자의 스토어에서 어떠한 사유로 해당 상품의 정보를 변경했을 수 있습니다.

    이런 경우, 해당 상품을 판매하고 있는 판매자에게 자세한 문의 부탁드립니다!

  • ♥찜한 데이터(PICK)가 갑자기 2개씩 중복 생성되었어요.

    앱을 완전히 종료하였다가 재실행해 주시기 바랍니다.

    혹시 앱 재실행 이후에도 동일한 현상이 지속될 경우 [마이페이지]-[앱 문의/건의하기]로 정보를 전달해주시면 빠르게 확인 도와드리겠습니다.

  • 댕구루에 입점하고 싶은데 어떻게 하나요?

    denguru_team@hiadone.com를 통해 신청해주시면 입점 조건에 해당하는지 검토 후 등록가능합니다.

  • 댕구루에 입점 신청했는데 승인 결과는 언제 나오나요?

    입점 심사는 주말, 공휴일 포함 약 10일 정도 소요될 수 있으며, 신청 시 보내주신 이메일 주소로 입점 결과가 발송됩니다.
